疫情项目化学习成果汇总

项目化学习 新冠
发布日期:   2020-07-13

国家突发公共卫生事件应急预案

前言

2020年初，新型冠状病毒在中国武汉爆发，随后蔓延至全球。经过两个月的不懈努力，中国疫情基本得到控制。为了使灾难不再发生，我院对疫情发生以来的种种事件进行了研究与总结，并撰写了《国家突发公共卫生事件应急预案》。

本预案选择了公共卫生事件中核心的四类人群即普通民众、医务工作者、政府决策者和病毒学家进行说明，对他们在事件中的行为进行了指导，使事件发生时社会能够有序、有效地应对，让人民群众的生命安全得到切实保障。

重大公共卫生事件是指某种传染病在短时间内发生、波及范围广泛，出现大量的病人或死亡病例，其发病率远远超过常年的发病率水平的情况。主要包括：

(1) 肺鼠疫、肺炭疽在大、中城市发生并有扩散趋势，或肺鼠疫、肺炭疽疫情波及2个以上的省份，并有进一步扩散趋势。

(2) 发生传染性非典型肺炎、人感染高致病性禽流感病例，并有扩散趋势。

(3) 涉及多个省份的群体性不明原因疾病，并有扩散趋势。

(4) 发生新传染病或我国尚未发现的传染病发生或传入，并有扩散趋势，或发现我国已消灭的传染病重新流行。

(5) 发生烈性病菌株、毒株、致病因子等丢失事件。

(6) 周边以及与我国通航的国家和地区发生特大传染病疫情，并出现输入性病例，严重危及我国公共卫生安全的事件。

(7) 国务院卫生行政部门认定的其他特别重大突发公共卫生事件。

本预案在重大公共卫生事件发生后由政府研判形势后得以启动，但其在平时也是一份富有建设性的建议，为应急体系建设提供了方向。

本预案依照《中华人民共和国传染病防治法》、《中华人民共和国食品卫生法》、《中华人民共和国职业病防治法》、《中华人民共和国国境卫生检疫法》、《突发公共卫生事件应急条例》、《国内交通卫生检疫条例》和《国家突发公共事件总体应急预案》制定。

主体

1. 日常卫生防护

1.1 病毒学家

在日常生活中，病毒学家大部分时间都用于研究病毒感染、致病的分子本质，为病毒基因工程疫苗和抗病毒药物的研制以及病毒病的诊断、预防和治疗提供理论基础及其依据的科学。 所以病毒学家应在日常研究过程中注意防控，避免病毒传出导致疫情复发。

（1）在研究病毒样品应做好全身防护，避免自身感染病毒；

（2）研究病毒样品应进行密封且保存；

（3）研究后的仪器应经过消毒后再保存或进行研究；

（4）产生的废物应用特殊垃圾袋保存密封后，且注明是生物垃圾；

（5）一经发现有生物危险的事物立即上报国家；

（6）发现某种药物对于某种长期流感有效立即上报国家；

（7）如若病毒不慎传播出去需立即上报国家。

1.2 医务工作者

在医院中每天要接触不同的病人，所以医务人员要加强防护，保证自身不被感染，才能去挽救更多的生命。

（1）在日常诊断时要佩戴口罩，医务帽。

（2）统一工作服，离开岗位要脱下。

（3）如有与血液有接触的工作，需佩戴手套。有外露伤口要遮挡，避免与化验血样有接触。

（4）注射器采用一次性注射器，注射前务必认真消毒。

1.3 普通民众

针对于普通民众，在平时的生活中需要养成良好的卫生习惯，这样可以使自己保持健康。还要保持良好的免疫能力。

（1）吃饭前和上完厕所后都需要认真洗手，保证手上没有病毒，为自己的安全提供保障。

（2）在医院时注意防护，以免细菌附着到人的身上。

（3）做好常用医疗物品的囤积，以备不时之需。

1.4 政府决策者

在日常中，政府应该加强卫生防护，在各个方面保障卫生，因为卫生是人民能否健康和经济体系是否稳定的基础。

具体措施：

（1）对食品、药品、公共环境卫生的监督管制以及相关的卫生宣传、健康教育、免疫接种等。

（2）对重大疾病尤其是传染病（如结核、SARS等）的预防、监控和治疗。

（3）建立社会机构来确保每个人都能达到健康生活的标准。

（4）建立完整、可靠的公共卫生体系。附：中国公共卫生体系

（5）定期举行防控演练，尤其是在春运等有大规模人员流动的时候。

2. 疫情来临时

2.1 病毒学家

在疫情来临时，应尽快研究病毒基因组的结构与功能，探寻病毒基因组复制、基因表达及其调控机制等，为后期的有效预防和疫苗研究做出便利。

（1）发现疫情应立即向国家报告；

（2）尽快研究病毒基因组的结构与功能，探寻病毒基因组复制、基因表达及其调控机制等，经严格考证后，再上报国家，便于查找疫情的源头；

（3）在做有关疫情研究时，发现某种药物对于疫情有效应多加科研后再对外声明，避免引起群众轰动；

（4）尽快总结病毒会引发的主要症状；

（5）在研究病毒中，若不慎感染病毒，立即隔离；

（6）尽快研究发现可以杀死病毒的物质，便于消毒预防

（7）研究出疫情源头后上报国家。

2.2 医务工作者

在此期间所有医务工作者都需更加警惕，绝不放过任何一名患者，减小二次传播的几率

（1）提高警惕不放过任何一个感染者。

（2）快速寻找病毒源头。

（3）尽快发现病毒特征，并上报国家，报告给民众，但要保持严谨。

（4）开会商讨解决方案，尽量出台解决方案。

（5）在于感染者接触时要佩戴防护服，鞋套，帽子。

（6）有能力或有志愿的可以支援病发案例较多地区

（7）维持医院秩序

（8）发布实时疫情消息。

2.3 普通民众

疫情来临时，首先需要保持冷静，不传谣不信谣。

  1. 做好食物储备，减少外出。

  2. 提醒身边的朋友，让他们也针对此次病毒做出准备。

  3. 并提前了解病毒的传播方式是什么，感染性有多强，致死率有多高等，保证自己不会被感染。

  4. 同时承担起自己的社会责任，保证社会基本运转。

  5. 服从国家管控，以大局为重。

2.4 政府决策者

政府在疫情来临时所作的决策决定了疫情是否会大规模传播，以及这次疫情会给经济带来怎样的影响。所以当疫情来临时，政府决策者所做的决定就变得至关重要。政府需要及时指派专家进行考察，并听取专家提出的防控建议。在2020年新冠疫情中，政府就是听取了赴武汉专家组的意见，对武汉进行了封城，这对疫情发展产生了重要的影响，下面这张图展示了管控开始时间与疫情流行程度的关系。

卫生部依照职责和本预案的规定，在国务院统一领导下，负责组织、协调全国突发公共卫生事件应急处理工作，并根据突发公共卫生事件应急处理工作的实际需要，提出成立全国突发公共卫生事件应急指挥部。

3. 疫情发生中

3.1 病毒学家

疫情中期，是传染及病毒变异的高发期，所以在这个时节是疫情防控最为重要的期间，在此期间病毒学家应更深层次地研究病毒，并研究出相关特效药。

（1）尽快研究出病毒核酸检测试剂；

（2）尽快研究出治疗方案和疫苗；

（3）协助医务工作者治疗患者；

（4）研究病毒是否发生变异，如若发生立即上报国家加强警戒；

（5）如有较好的预防方法可向国家提出；

（6）加紧对于特效药的研究；

（7）发现特效药，应多进行试验，且研究此特效药的副作用；

（8）依照每日疫情数据，客观推出疫情拐点。

3.2 医务工作者

更多的病患出现了，下面要做的就是治疗好每一位患者，因为政府已经组织隔离等措施，在阻止病患增加。

（1）应用治疗方案，若可以公之于众，如不可以继续研究方案，时间就是金钱和人命。

（2）接触病危或病情严重者，要佩戴呼吸面具，防护服，手套，脚套等，保证身上没有任何一处裸露。

（3）时刻保持警惕，防止病毒变异或进化。

（4）维持医院秩序

（5）坚持每天消毒。

（6）若其他国家也出现了病患，积极支援，记住”人类命运共同体。”

（7）发布实时疫情信息。

3.3 普通民众

疫情发生时，群众的行动是战胜疫情的关键。要为抗击疫情贡献出自己的一份力量，有条件的可适当捐款，家中不富裕的也可以在精神层面上支持抗”疫”工作。

（1）尽量不去人群密集的场所，如实在没有办法避免请戴口罩。（口罩请选择N95或医用口罩）

（2）勤洗手，在外出接触过公共设施、呼吸道分泌物、探访医院后要洗手。

（3）不要随地吐痰。打喷嚏或咳嗽时用纸巾或袖肘遮住口、鼻。

（4）加强锻炼，作息规律，保持室内空气流通。

（5）避免接触野生禽畜。

（6）从疫区外出归来，如发现发热咳嗽等呼吸道感染症状，应及时配戴口罩就医。

3.4 政府决策者

疫情发生中正是病毒传播最快的时候，会遇到更多的问题，这就更考验政府的决策力和领导力。

具体措施：

（1）划定疫点、疫区并进行卫生处理。

（2）对相关场所或场所内的一定区域的人实施隔离。

（3）实施交通管制或其他控制措施。

（4）启用财政预备费和储备的应急救援物资，必要时调用其他急需物资。

（5）保障基本生活必需品的供应。

（6）严惩囤积居奇、哄抬物价、制假售假等扰乱市场秩序的行为。

（7）严惩哄抢财务、干扰破坏应急处置工作等扰乱社会秩序的行为。

（8）紧急调集人员或者临时征用动产或不动产。

4. 疫情结束后

4.1 病毒学家

疫情之后，为避免疫情卷土重来，我们需要从疫情之中总结经验，并避免再次犯同样的错误，存有病毒样本的实验室要妥善保管好样本，加快研究好疫苗。

（1）存有病毒样本的实验室妥善保管好病毒样品；

（2）加快研究有效疫苗；

（3）从疫情中总结经验与教训。

4.2 医务工作者

结束后也不能松懈，谨防向”郑州毒王”这样的案例出现，要等到国家或世界清零才可以放松。

（1）每天对医院内病人进行核酸检测。

（2）对与感染病人有接触的医生进行检测和消毒。

（3）坚持每天对医院进行消毒。

（4）医生做好每天的防护。

（5）处在疫区的医院要增加消毒次数。

（6）将实时消息发布。

（7）做好对凯旋归来的医生的奖励工作。

4.3 普通民众

疫情结束后，首先要及时的接种疫苗，确保以后自己不会拥有再次染上该病毒的风险。其次，我们需要努力工作，确保在下次疫情来临时有经济能力去生存。最后，我们需要从本次疫情中得到经验，如：敬畏生命等。

4.4 政府决策者

疫情结束后，政府官员应该进行反思，总结经验，争取做得更好。还要做好善后工作和经济的发展。

具体措施：

（1）制定出台救助、抚慰、抚恤、安置等善后工作计划。

（2）对做出显著成绩和贡献的单位和个人给予表彰或者奖励；对致病、致残、死亡的工作人员给予补助、抚恤。

（3）对符合条件的一线医疗卫生人员给予适当的津贴。

（4）对举报有功的单位或个人给予奖励。

（5）给予疫情防控期间紧急调集的人员合理报酬，征用公民动产或不动产的，予以返还或补偿。

（6）帮助中小企业复工复产。

（7）地方各级人民政府应组织有关部门对应急处理期间紧急调集、征用有关单位、企业、个人的物资和劳务进行合理评估，给予补偿。

结语

疫情防控是一场没有硝烟的战争，在这个全球化时代，局部的公共卫生事件处理不善就会变成全球性公共卫生事件，让全人类面临危险。当事件发生后，各国需及时研判形势并果断行动，启动此预案并以强有力的手段全面执行，全力保障人民生命安全。当疫情稳定后，由政府决定终止此预案，并进行善后工作。

外交部发言人行动指南

行动宗旨

发言人的行动宗旨是中国立场、基于事实、精准传达、平等交流，维护中国的形象与利益，为增进中国与世界的相互了解和理解而努力。

行动指南

首先，发言人代表中国政府，所以一切的行为都要基于中国的立场。当事件有利于中国利益，我们应当予以感谢；反之当事件不利于中国利益，我们应予以回击。同时，我们还要基于中国当下国际关系进行回应。

近日，有一种”中国商品携带病毒“的说法，对此，外交部发言人耿爽进行回应称：疫情发生以来，看到一些人不负责任地散播谣言制造恐慌。中国是制造业大国，也是防疫和医疗物资生产和出口大国。当前疫情在全球蔓延，很多国家面临口罩、防护服、呼吸机等防疫物资缺口，希望得到中国援助或从中国购买。如果有人说“中国制造有毒”，那么请说这种话的人面对新冠肺炎疫情肆虐，不要戴中国制造的口罩，不要穿中国生产的防护服，不要用中国出口的呼吸机，以免染上病毒。这段话即维护了中国的商业利益，还对有关的人进行了有利的回击。

同时，发言人的回应要基于事实。在疫情中，发言人要向外界介绍中国所做的事情，促进中国与世界的相互理解。

3月20日，外交部发言人耿爽例行记者会上介绍道：中国政府已经宣布向82个国家和世界卫生组织、非盟提供援助。当前，我们也在向遭受疫情的国家伸出援助之手，主要从四个方面做出努力：一是在政府间援助方面，二是在医疗技术合作方面，三是在地方政府援助方面，四是在民间援助方面。很多中方对外提供的援助物资都写有寄语，既有“千里同好，坚于金石”等中国古语，“道不远人，人无异国”等韩国诗句，也有“我们是同海之浪，同树之叶，同园之花”等意大利哲学家名言。这表达了中国人民与世界各国人民加强抗疫国际合作、共同推动构建人类命运共同体的心声，提高了中国的国际形象。

发言人还要对相关事件进行精准的传达，让一些重要措施尽早详实地公之于众。

3月1日如疫情进一步发展，外交部领事司司长崔爱民表示，如疫情进一步发展，将协助安排华侨、留学生回国。这一发言让海外的中国公民吃了一粒“定心丸”，让他们感受到祖国这一坚强后盾。

外交部发言人还要与广大记者进行平等的交流，做好国际。在疫情中，国际间更应该积极交流，全球共同抗“疫”，让各国渡过难关。

新冠疫情的真面目

新年本来是一家人阖家团圆的日子，但是一场突如其来的瘟疫改变了这份和谐。2020年年初，新型冠状病毒疫情在湖北武汉爆发，之后全国蔓延。在国家的积极防控下，湖北以外的疫情得到初步控制。经过不断地研究，我们对新型冠状病毒有了初步的了解。

什么是新型冠状病毒？首先，冠状病毒是自然界广泛存在的一大类病毒，仅感染脊椎动物，如人、鼠、猪等，它是RNA病毒基因组最大的病毒。因为在显微镜下这种病毒像王冠，所以得名冠状病毒。新型冠状病毒是最新发现的一种冠状病毒，也是已知第七种可以感染人类的冠状病毒，发源地和宿主还不明。新型冠状病毒可以引起肺炎，最新研究表明，它还会攻击中枢神经，危害人体健康。

这种病毒是怎样造成传染的呢？新冠病毒可以以各种方式进入宿主机体，且传播形式多样。它主要以飞沫传播、接触传播和气溶胶传播为主要传播途径，尤其以飞沫传播速度最快，传播面积最广。它可以它相对于SARS来说死亡率不高，约为3.43%，平均每天会出现972例新冠状病毒感染者^[2]^（这些数据截止至2020.2.24.）。由于它传染性较强，又赶上了”春运”，短短两个月让中国各省都出现了感染病例。

新冠状病毒之所以会引起肺炎，是因为病毒会通过人体气管到达肺泡，引起人体自身的免疫反应，这有可能会引起一种连续、剧烈的免疫反应———细胞因子风暴，可能引起大量的体液积聚进而导致肺部阻塞，使患者因无法正常呼吸导致窒息而亡。同时，大量的血液可能流到体液之中，造成患者因失血休克死亡。

那么有什么方式能治疗新冠肺炎呢？目前还没有针对新冠肺炎的有效药物，以对症支持治疗为主。它的原理很简单，当患者出现 ARDS （即急性呼吸窘迫综合症，简称 ARDS）时无法自己呼吸，他们的肺功能也会衰竭，等不到人体自行消炎，往往已经死亡。这时，就要通过各种手段和病毒”赛跑”，在炎症自行消失之前，维持患者的生命，这就是对症支持治疗。另外，康复者血浆治疗和中西医结合治疗也取得了很不错的效果，目前正在推广。

现在，人们应该做好防控措施，严防感染。以下几种防控方法能很好地防止感染。

少出门、不去人群密集的场所

新冠病毒主要的传播方式为飞沫传播，所以我们少出门、少接触人群就可以降低感染几率。实在无法避免的外出活动请佩戴医用口罩或N95口罩。

注意个人卫生、尽量避免接触

在外出接触过公共设施后要洗手并用酒精消毒。不与他人接触，避免接触传播。

加强个人免疫力，居家也要防疫

居家时间要加强锻炼，作息规律，加强免疫力。保持室内空气流通，防止气溶胶传播。

进行自我健康监测

如果你来自疫区请自觉在家隔离，如有发热、咳嗽、乏力等新冠肺炎感染症状，应及时上报就医。

在抗击新冠肺炎的过程中，出现了很多谣言和阴谋论。相似地，病毒是控制宿主细胞核酸为自己所用，谣言就是控制我们的大脑为造谣者所用。我们一定要保持独立思考，不传谣、不信谣，为抗击疫情提供良好的环境。

在国家的积极防控和社会各界，疫情防控逐渐显露成效，医疗资源逐渐得到满足，患者得到及时救治，胜利已经不远。

疫情终将过去，春天必将到来。了解了新冠肺炎疫情的真面目，我们更应该加强防范，早日打赢疫情防控阻击战。

生命的意义

“每个人的最终结局都是死亡，那么为什么我们还要活着？”

人的生命是分为两个维度存在的——肉体及这个肉体上承载的一些东西，有些人称之为灵魂。此处的灵魂并非一些宗教中所指，而是人包括性格、情感、言论、事迹等的统称。人一生中遇到的所有人组成了他的一生，他或多或少都会存在于与他相遇的人的意识里。的确，每个人的肉体都会离去，但这不意味着“最终结局”，生命的一部分还可以在别人的意识中存留在这个世界，只有彻底被人遗忘，才是真正死了。

然而，除非成为改变历史的伟人，我们很难被人世代铭记，于是几代之后，渐渐被忘却。此时，我们迎来了真正的死亡吗？对于那些实现了自我价值的人，答案是否定的。

如果我们活着时，为了自己族群、国家、物种乃至文明的发展和延续做出贡献，用生命塑造、融入和传承人类的精神，那么作为这个精神的融入者或是塑造者，我们也就会直接或间接地存在于世上。这或许是我们大多数普通人实现自我价值的最好方式。

这次疫情中，那些热爱祖国，热爱生命，热爱真理，热爱奉献的人，做出了很多平凡而伟大的事：有钟南山院士临危受命前往武汉调查疫情，有张定宇院长身患渐冻症依然坚守一线，有李文亮、夏思思等医生以身殉职，有支援湖北医疗队立下“不计报酬，无论生死”的誓言后奔赴前线救死扶伤，更有全国各行各业众志成城全力支持抗疫、全国人民被动员起来自我防护疫情。这次抗疫的种种事迹几十年后可能会被遗忘，但无数普通人在此次疫情中创造出的精神将会成为我们世代传承的精神的一部分，随之他们生命的“最终结局”也不再是死亡。

“有的人活着，他已经死了；有的人死了，他还活着”。碌碌无为之辈，在肉体消亡之前，灵魂已经被忘却，于是注定走向死亡；实现了自我价值的人，虽然肉体消亡，但灵魂却融入人类精神之中，实现“永生”。所以，朋友！既然生命的最终结局不一定是死亡，我们还有什么不努力地生活呢？我们还有什么理由不热爱我们的生命呢？
